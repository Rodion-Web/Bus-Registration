## 📌 Описание

Веб-приложение для регистрации пользователей на автобусные рейсы с учетом ограничений мест, валидацией данных и защитой от дублирующих записей.

Приложение реализует бизнес-логику и обеспечивает безопасную работу с пользовательскими данными.

---

## ⚙️ Функционал

* Регистрация пользователя (ФИО, телефон, выбор автобуса)
* Валидация данных
* Предотвращение дубликатов
* Контроль количества мест по каждому автобусу
* Отображение доступных мест в реальном времени
* Отображение персонала (staff)
* Экспорт данных в CSV

---

## 🔒 Безопасность и приватность

* Персональные данные (ФИО, телефоны) **не отдаются через публичный API**
* Все данные хранятся локально в папке `/data`
* Папка `/data` исключена из Git через `.gitignore`
* Доступ к `/data` закрыт на уровне сервера
* Публичный API возвращает только агрегированные данные (количество мест, лимиты)

---

## 🧠 Backend

* PHP (без фреймворков)
* REST API:

  * `GET /api.php?action=state` — публичные данные (без персональной информации)
  * `POST /api.php?action=submit` — регистрация пользователя
* Хранение данных в JSON
* Генерация CSV-файла
* Защита от race condition (`flock`)

---

## 🎨 Frontend

* HTML, CSS, Vanilla JavaScript
* Адаптивный интерфейс
* Работа с API через `fetch`
* Валидация на клиенте
* Форматирование телефона
* Уведомления (toast)

---

## 🗄️ Хранение данных

* JSON — внутреннее хранилище (не публичное)
* CSV — формат для экспорта

## 🚀 Запуск

### Требования

* PHP 8+

---

### Локальный запуск

```bash id="ru_run"
git clone https://github.com/Rodion-Web/Bus-Registration.git
cd Bus-Registration
php -S localhost:8000
```

Открыть в браузере:
http://localhost:8000

---

### 💡 Примечания

* База данных не требуется
* Файлы данных создаются автоматически при первом запуске
* Подходит для быстрого локального тестирования
